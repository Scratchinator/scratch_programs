# Deep Sea


![](preview.png)

In this game you start small and have to work your way up by eating smaller animals, but be careful there are also larger animals that want to eat you.

## Key assignment

| Key | description |
| --- | --- |
| W | Up
| S | Down
| A | Left
| D | Right
| 🔼️ | Up
| 🔽️ | Down
| ◀️ | Left
| ▶️ | Right
